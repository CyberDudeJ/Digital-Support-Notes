4/5 layer model <br />
Protocol suite that pre-dates the OSI model <br />

:::tip Difference between TCP and OSI
The TCP model combines application, session and presentation layer together into the `application layer`
:::


**Layers:** <br />
1. **Physical**
    - Cables, NIC
    - Everything you can touch
2. **Data Link**
    - Ethernet, Switches
3. **Network**
    - IP, Routers
4. **Transport**
    - TCP, UDP
5. **Application**
    - HTTP, FTP, SMTP
